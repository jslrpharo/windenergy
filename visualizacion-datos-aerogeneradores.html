Data Visualisation & Processing for Wind Turbine Data

ACMSL provides tools for visualising, processing and exporting time-series signal recordings from simulator sessions and from field dataloggers — enabling deeper analysis beyond what is visible in real-time during operation.

6-1  LdP Datalogger Viewer Application

The LdP Datalogger Viewer is a Windows application for visualising signal evolution recordings captured during sessions with any ACMSL simulator — or from the DL1300 industrial datalogger.

Main capabilities:

  • Multi-channel time-series display – visualise tens of signals simultaneously with zoom, pan and cursor tools
  • CSV export – export any selection of signals to CSV for further analysis in Excel, Python, MATLAB, etc.
  • Resampling – resample signals to a uniform time base before export
  • SQLite backend – recording files use SQLite, allowing direct queries and external data processing
  • Multi-session comparison – overlay recordings from different sessions for comparative analysis

Available as a free download. Includes integrated help documentation in 6 languages.

6-2  Visualising CSV, Excel and Other Time-Series Files

Beyond simulator recordings, the LdP Datalogger Viewer can open and visualise data from standard CSV files and other tabular formats, making it useful for:

  • Post-processing SCADA data exports from operational wind farms
  • Analysing data collected by the DL1300 industrial datalogger
  • Visualising time-series from laboratory instrumentation
  • Comparing design models with measured field data
