All ACMSL software includes a free 2-day evaluation license, granted automatically on first launch — no credit card, no prior registration required. Install, run, and start learning.

How the Evaluation License Works

  1. Download and install the package on a Windows PC
  2. Launch the application for the first time
  3. Fill in your name and email when prompted
